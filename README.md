# three.js - misc - octree collisions

Visor web sencillo hecho con **Three.js** para cargar y visualizar un modelo de escenario con controles de movimiento.

---

## ✨ Qué incluye

- Renderizado 3D en navegador con **Three.js** y un ejemplo del apartado **games / fps**. 
- Controles en **primera persona** basados en **PointerLockControls**, donde el mouse controla la vista y el teclado permite el desplazamiento. 
- Movimiento con teclas **W, A, S, D**, salto con **SPACE** y vista libre con **MOUSE**, siguiendo la interacción del ejemplo oficial. 
- Escenario 3D interactivo orientado a una experiencia tipo **FPS**, pensado para recorrer la escena en tiempo real. 
- Integración de una estructura base en HTML para cargar el contenedor de la escena, instrucciones en pantalla y el módulo principal de JavaScript.
